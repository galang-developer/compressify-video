### **Persyaratan**  
- Node.js (v18 atau lebih baru)  
- NPM / Yarn / PNPM / Bun  

### **Instalasi**  
1. **Clone repository**:  
   ```bash
   git clone https://github.com/galang-developer/compressify-video.git
   cd compressify-video
   ```  

2. **Install dependencies**:  
   ```bash
   npm install
   # atau
   yarn install
   # atau
   pnpm install
   ```  

3. **Jalankan Aplikasi**:  
   ```bash
   npm run dev
   # atau
   yarn dev
   ```  
   Buka [http://localhost:3000](http://localhost:3000) di browser.
